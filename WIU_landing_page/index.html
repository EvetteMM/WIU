About World Surf Project

The World Surf project is a practical platform for the web development student to explore user interface and usability through design, anonymous testing and the gathering of feedback from end users. Using Adobe Illustrator, the student will first develop both desktop and mobile wireframes for the initial layout and design of a website. Each wireframe will be designed with user-friendly functionality as its objective and will demonstrate step-by-step a userâ€™s progression through the site. The goal in this design phase is to create intuitive icons, inputs and buttons that require as little explanation as possible in order to meet all of the criteria for Usability Principles - Learnability, Efficiency, Memorability, Errors and Satisfaction.

These wireframes will be uploaded to www.navflow.com, a site that not only tests wireframe usability but also gives the designer an opportunity to gather anonymous feedback by generating a testing link for users. The program allows you to create 'hotspots' for items like input boxes, directional icons and buttons to generate feedback about how the wireframe is being used. Navflow members have the opportunity to take random tests on Navflow that generate click feedback as well as verbal feedback. Finally, Navflow gives you the ability to test target markets through social media venues like Facebook, Twitter, LinkedIn and GooglePlus.
